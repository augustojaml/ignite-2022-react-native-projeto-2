<h1  align="center">Como iniciar</h1>

```bash
  git clone https://github.com/augustojaml/ignite-2022-react-native-projeto-2.git
```

<h1  align="center">Instalar dependências</h1>

```bash
  yarn start
```

<h1  align="center">Executar app</h1>

```bash
  expo start
```
